= Ruby 101

Este proyecto contiene todos los archivos del taller de Ruby 101 presentado en Monterrey, el 23 de Febrero del 2011 para el Rubyton.

Dentro de cada Folder viene un archivo con el nombre de <b>readme.rb</b> donde viene la explicaciÃ³n del contenido que tiene esa parte del taller.

Se hace esto de manera que el tallerista pueda explicar la sintaxis de Ruby desde el archivo utilizando como ventaja el highlighting del editor de texto de su preferencia.

== Requerimientos

Instala ruby > 1.8.7

Instala bundler
	gem install bundler
	
Corre el comando bundle
	bundle
